## üöÄ Running Examples

Each example is self-contained. To run:

1. Create a new React project (Vite, Next.js, etc.)
2. Install the editor: `npm install @bosonbrain/editor`
3. Copy the example code
4. Import the styles in your main file
5. Run your dev server

## üí° Tips

- Always import the styles: `import '@bosonbrain/editor/styles';`
- The editor is fully controlled - you don't need to manage any state unless you want to
- Use TypeScript for the best development experience
- Check the main README for full API documentation
